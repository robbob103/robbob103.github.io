  Question.
Does the gas light, but then go out?
 
Answer.
If it is the piezo version with a pilot then does the pilot light and is it a good flame that reaches the thermocouple? If not then the pilot jet is likely to be blocked and we recommend a new pilot assembly.
 
If the pilot is ok or it is the standard version without a pilot, then the likely problem is the thermocouple. It is worth checking to see if the nut connecting it to the gas valve hasn’t worked loose. Please note that it should not be over tightened.
 
If a new one is required, please say when ordering if it is a thermocouple for a piezo boiler (F20/P, F44/P)or non-piezo boiler (F20, F44).
 
Question.
Does the flame seem low when on full setting?
 
Answer.
It is very likely that the main burner jet has become blocked and it needs servicing.
IF THE FLAME IS VERY YELLOW OR PRODUCING SOOT THEN THE APPLIANCE SHOULD BE TURNED OFF AND SERVICED BEFORE FURTHER USE.
 
Question.
Does the water tap drip or not work properly?
 
 
Answer.
It is worthwhile checking to see if any scale has built up on the rubber washer. This is easy to do when the boiler is empty of water and cooled down.
Just unscrew the plastic part of the tap and the rubber seating cup can be pulled off and washed. Check to see if the spindle and spring are intact. If OK then push the seating cup back on firmly and replace on the metal tap body. If this does not resolve problem then we can supply a new tap top.
